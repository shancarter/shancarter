About

I’ve worked at newspapers my entire professional life and I currently design and program interactive graphics for The New York Times. In the past I made graphics for the paper edition, but now I mostly make things for the website with HTML and Javascript, mostly using D3. I also dabble in Processing, R and using Python to script Maya. I generally don’t get worked up about things. I prefer Illustrator to Photoshop.
